Excel in 2026 has seen a massive shift toward Agentic AI and seamless Cloud-Web integration. Most updates revolve around making data processing more autonomous and reducing the technical barrier for complex formulas.


Here are the key new features and updates for Excel as of March 2026:

🤖 The Rise of "Agent Mode"
The most significant update is the full rollout of Agent Mode within Microsoft 365 Copilot. Unlike the standard chat sidebar, Agent Mode is designed to be "hands-off."

Autonomous Workflows: You can set a "destination" (e.g., "Clean this sales data and create a monthly summary dashboard"), and the Agent will map out the steps and execute them across multiple sheets.


Model Switcher: Users can now toggle between different LLMs (like OpenAI's GPT models or Anthropic's Claude) directly within Excel to see which performs better for specific logic tasks.


Local File Support: Copilot can now query and analyze modern Excel files ($ .xlsx$, .xlsm) stored locally on your device, not just those saved on OneDrive or SharePoint.


📊 New Functions & Data Handling
Microsoft has introduced several functions to simplify getting data into the grid without needing complex Power Query setups for every task:


IMPORTCSV & IMPORTTEXT: These new functions allow you to pull external data directly into a refreshable dynamic array using a single formula. If the source file changes, a simple "Refresh All" updates your grid.


Clean Data Button: A new native tool that uses AI to detect inconsistencies (like mixed date formats or duplicate records) and fixes them with a single click.


PivotTable Auto-Refresh: Pivot tables can now be set to refresh automatically whenever the source data changes, eliminating the need for manual refreshes or VBA macros.


🛠️ Error Handling & UI Improvements
Descriptive Error Cards: No more guessing why a formula shows #VALUE!. When you hover over an error, a card now appears explaining the logic failure in plain English and suggesting actionable steps to fix it.

Formula Auto-Complete: An AI-powered engine that predicts the rest of your formula as you type, significantly faster and more context-aware than the old IntelliSense.


PivotTable "Spill": Pivot tables now support dynamic arrays, allowing them to "spill" into adjacent cells more naturally, making them more compatible with other modern functions like SORT and FILTER.

🌐 Excel for Web Power-Up
The Web version of Excel has reached parity with many Desktop features:

Full Power Query: The full Power Query Editor and Import Wizard are now available directly in the browser.


Office Scripts Enhancements: You can now use Office Scripts to automate tasks like "Save as PDF" and "Email to Team" directly from the web client.
